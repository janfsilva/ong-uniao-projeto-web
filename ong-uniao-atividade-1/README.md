# Entrega 1 – Fundamentos e Estruturação

## Objetivo
Construir a base do site da **ONG União** usando **HTML5 semântico**, com páginas e seções bem organizadas.

## Estrutura de Arquivos
- `index.html` – página inicial (Quem Somos)  
- `projetos.html` – página com os projetos da ONG  
- `cadastro.html` – página com o formulário de cadastro  
- `imagens/` – imagens otimizadas (JPG, PNG e SVG)

## Tecnologias Utilizadas
- HTML5 (tags semânticas, `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`)
- Atributos de acessibilidade básicos (`alt` nas imagens)

## Descrição do Desenvolvimento
Foi criada a estrutura completa do site com conteúdo textual, imagens e formulário, seguindo boas práticas de semântica e organização.  
**Sem CSS e sem JavaScript nesta etapa** — foco somente em HTML e acessibilidade básica.

## Observações
Os links de CSS e JS não estão presentes nesta entrega por exigência do escopo da Atividade 1.
