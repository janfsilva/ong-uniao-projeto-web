# Entrega 3 – Interatividade e Funcionalidades

## Objetivo
Adicionar **JavaScript** para tornar o site interativo, com foco na **validação simples do formulário** de cadastro.

## Estrutura de Arquivos
- `index.html`, `projetos.html`, `cadastro.html`
- `css/style.css`
- `js/script.js` – validação simples do formulário (checkValidity + alertas)
- `imagens/`

## Tecnologias Utilizadas
- HTML5, CSS3, JavaScript (ES6 básico)

## Descrição do Desenvolvimento
Implementado script que:
- impede envio se houver campos obrigatórios vazios;  
- mostra alerta amigável ao usuário;  
- simula envio bem-sucedido e limpa o formulário.

## Observações
A validação é do lado do cliente e demonstra uso básico de JS nesta etapa.
