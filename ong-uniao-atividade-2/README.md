# Entrega 2 – Estilização e Leiautes

## Objetivo
Aplicar **CSS3** para estilização visual do site da **ONG União**, mantendo a estrutura HTML da Entrega 1.

## Estrutura de Arquivos
- `index.html`, `projetos.html`, `cadastro.html`
- `css/style.css` – estilos principais (menu em linha, tipografia, cores, responsividade)
- `imagens/` – imagens otimizadas

## Tecnologias Utilizadas
- HTML5
- CSS3 (layout responsivo, imagens fluidas, menu horizontal)

## Descrição do Desenvolvimento
Foram definidos cores, tipografia, espaçamentos e layout responsivo.  
As **imagens foram configuradas como responsivas** e o **menu navega em linha** no desktop (e empilha no mobile).

## Observações
Sem JavaScript nesta entrega (foco somente em HTML + CSS).
